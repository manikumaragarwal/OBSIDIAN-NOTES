# Types of IMAGES :

- main 4 types :
  1. JPG IMAGES - _for normal images_ (no tranparency)
  2. GIF IMAGES - _for animated images_
  3. PNG IMAGES - _for images with transparency_ (high file size)
  4. SVG IMAGES - _vector form illustration_ (responsive)
