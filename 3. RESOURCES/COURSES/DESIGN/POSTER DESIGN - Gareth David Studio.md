# Anatomy of Poster Design
---
## 1. Foundation
- **Surface Area**
- **Margin** : 
	- Enhance readability
	- More -> extreme posters
	- less -> minimal posters
- **Grid** : 
	- provides structure.
	- Types : 
		- column
		- baseline
		- modular
		- hierarchical
		- diagonal
		- axial 
		- radial
## 2. Visual Elements
1. **Color** :
	- select the mood.
2. **Shape** :
	- Visual cues : bring attention towards something.
3. **Image** :
4. **Typography** :
5. **Surface Texture** :
6. **Iconography** : small elements to add simplicity.
## 3. Compositions
1. **Visual Hierarchy** : 
	- prioritises information, clear and effective communicatoin.
2. **Primary Hook** : 
	- attention magnet
	- image, typography(display font), etc.
3. **Secondary Hook** :
	- compliment primary hook
	- images, typography : tagline, subheading, or intriguing detail.
4. **Supporting element** : 
	- backbone. 
	- typography, paragraphs and pullouts, iconography, textures, borders, patterns, etc.
	- bring cohesivenes.. 
5. **Space** :
	- let person breathe
	- enhace clarity and aesthetics. 
6. **Form** :
	- overall visuall element's arrangement.
	- add depths, character and movement.
	- Call To Action.
7. **Call to Action** :
	- catalst of engagement.
		-  passive viewers into active participants.
8.  **Footer Elements** : 
	- can be put anywhere in the poster (not only footer.)
	- on margin space or Fringe of poster design.
