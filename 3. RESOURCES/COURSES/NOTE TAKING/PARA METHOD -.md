# PARA
### Projects
- Anything that requires completion of multiple tasks, that you need to complete, and all are contributing towards a certain goal.
- **Time span** : 2-3  months. 
-  It contains all the files categorized in groups, required for a particular Project. 
- eg : **MAKING A WEBSITE**
	- **Platform** : which platform you're using to host the site. 
		- all the different types of services.. 
		- pricing comparisons
		- etc.
	- **Design** : what theme or design your site going to have.
		- Different Themes comparison and their links
		- Different Color pallets
		- UI elements
		- UI/ UX designsDesign
	- **Write** : what texts is going to be there in the site.
		- All the ideas
		- etc.
### Areas
- Something you're Responsible for longer period of time.
- Areas are something that goes on-and-on, it's doesn't stops after some time.
- Areas can be consist of many Projects. 
- Buying an House (PROJECT) -> life (AREA)
- eg : 
	- Health : contains everything like : fitness, diet, medications, relationships, sleep
	- 
### Resources
- Collection of information, that can be usefull in the future. 
- Not necesarily usefull now. 
- All hobbies go in here.
- All notes / links / things that you come across during content consumptions.
- Eg :  notes on 
	- Movies / songs / video lists.
	- some courses I'm taking
	- Books I am readings. 
	- random thoughts. 
### Archive
- Instead of deleting Notes, keep them in the Archive.
